# **App Name**: DigitalNest

## Core Features:

- Landing Page: Displays information about DigitalNest, including its value proposition, core SaaS modules, and a call to action.

## Style Guidelines:

- Primary color: A vibrant orange (#FF7F50) to reflect the energy and dynamism of the African digital market.
- Background color: A light beige (#F5F5DC) to provide a neutral and clean backdrop that is gentle on the eye.
- Accent color: A deep blue (#4682B4) to highlight key actions and important information, creating contrast and focus.
- Body font: 'PT Sans', sans-serif, for clarity and readability in both headlines and body text, contributing to a modern yet approachable design.
- Clean, modular layout with clear calls to action. Prioritize a user-friendly experience suitable for both technical and non-technical users.
- Use a set of consistent and recognizable icons, tailored to each of the core SaaS modules like NestMall and NestTravel, to enhance visual understanding and navigation.
